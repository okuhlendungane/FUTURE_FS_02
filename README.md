📊 Mini CRM – Client Lead Management System
📌 Overview

This project is a Mini CRM (Client Relationship Management System) built as part of my Full Stack Web Development internship with Future Interns.

The system is designed to simulate how businesses manage leads coming from their websites — allowing admins to capture, view, and update customer information efficiently.

⚠️ Note: This version focuses on frontend functionality only (HTML, CSS, JavaScript). Backend and database integration will be added in future iterations.

🎯 Features
* Admin Login Interface
:Simple login UI to simulate secure access
* Add New Customer / Lead
:Capture customer details such as name, email, and source
  *  Lead Dashboard
:Display all added leads dynamically in a table
* Update Lead Status
:Change customer status (e.g. New → Contacted → Converted)
* Dynamic UI Updates
:Uses JavaScript to update the DOM without page reloads
* Tech Stack
* HTML5 – Structure
* CSS3 – Styling & layout
* JavaScript (Vanilla JS) – Functionality & DOM manipulation
*Key Learnings
:Understanding how CRM systems manage client data
Working with DOM manipulation to dynamically update UI
Structuring a simple admin dashboard interface
Implementing real-world workflows (lead tracking & status updates)
* Limitations (Current Version)
* No backend (Node.js / Express not yet implemented)
* No database (data is not persisted after refresh)
* Authentication is UI-based only (not secure yet)
* 🔮 Future Improvements
🔗 Backend integration (Node.js + Express)
🗄️ Database (MongoDB / MySQL) for persistent storage
🔐 Proper authentication system
🔍 Search & filter functionality
📊 Basic analytics (total leads, conversions)
